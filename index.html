Á minha 1º página com bootstrap.

Como apliquei o Bootstrap na minha página?

É bastante simples para falar a verdade...

Primeiro para o css eu fui ao site do Bootstrap e peguei o cdn mais recente (28/02/2022), depois integrei-o colocando a importação do bootstrap(css) no <head>, como mostrado:

<head>
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous">
</head>

E finalmente para o javascript eu fiz o mesmo procedimento porém utilizando o cdn do Bootstrap (javascript) e colocando-o no body do meu website, como mostrado:

<body>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-ka7Sk0Gln4gmtz2MlQnikT1wXgYsOg+OMhuP+IlRH9sENBO0LRn5q+8nbTov4+1p" crossorigin="anonymous"></script>
</body>

E foi desta forma que eu apliquei Bootstrap no meu website(O website que está a ser agora visto).
